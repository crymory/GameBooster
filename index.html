Выберите скрипт оптимизации

Мы предлагаем набор скриптов для максимальной оптимизации вашей системы под игры

Установит все необходимые программы и драйверы для оптимальной работы вашего ПК.

Install-GameApps -InstallType Full -IncludeDrivers $true -CleanupAfterInstall $true
Скрипт скопирован! Вставьте его в PowerShell для запуска.

Отключит ненужные службы Windows для освобождения системных ресурсов.

Optimize-GameServices -DisableWinDefender $true -DisableTelemetry $true -DisableWindowsUpdates $false -OptimizeStartup $true
Скрипт скопирован! Вставьте его в PowerShell для запуска.

Оптимизирует настройки реестра Windows для максимальной производительности в играх.

Set-GameRegistry -OptimizeInput $true -OptimizeGraphics $true -DisableVisualEffects $true -PrioritizeGames $true
Скрипт скопирован! Вставьте его в PowerShell для запуска.

Оптимизирует сетевые настройки для минимального пинга и стабильного соединения.

Optimize-GameNetwork -SetOptimalMTU $true -OptimizeTCP $true -SetDNS "1.1.1.1,8.8.8.8" -FlushDNSCache $true
Скрипт скопирован! Вставьте его в PowerShell для запуска.

Выполнит полную оптимизацию системы, включая все вышеперечисленные настройки.

Invoke-Expression (New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/rage-optimizer/GameBooster/main/full-optimization.ps1')
Скрипт скопирован! Вставьте его в PowerShell для запуска.
